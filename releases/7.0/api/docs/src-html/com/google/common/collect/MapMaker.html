039    /**
040     * <p>A {@link ConcurrentMap} builder, providing any combination of these
041     * features: {@linkplain SoftReference soft} or {@linkplain WeakReference
042     * weak} keys, soft or weak values, timed expiration, and on-demand
043     * computation of values. Usage example: <pre>   {@code
044     *
045     *   ConcurrentMap<Key, Graph> graphs = new MapMaker()
046     *       .concurrencyLevel(32)
047     *       .softKeys()
048     *       .weakValues()
049     *       .expiration(30, TimeUnit.MINUTES)
050     *       .makeComputingMap(
051     *           new Function<Key, Graph>() {
052     *             public Graph apply(Key key) {
053     *               return createExpensiveGraph(key);
054     *             }
055     *           });}</pre>
056     *
057     * These features are all optional; {@code new MapMaker().makeMap()}
058     * returns a valid concurrent map that behaves exactly like a
059     * {@link ConcurrentHashMap}.
060     *
061     * The returned map is implemented as a hash table with similar performance
062     * characteristics to {@link ConcurrentHashMap}. It supports all optional
063     * operations of the {@code ConcurrentMap} interface. It does not permit
064     * null keys or values.
065     *
066     * <p><b>Note:</b> by default, the returned map uses equality comparisons
067     * (the {@link Object#equals(Object) equals} method) to determine equality
068     * for keys or values. However, if {@link #weakKeys()} or {@link
069     * #softKeys()} was specified, the map uses identity ({@code ==})
070     * comparisons instead for keys. Likewise, if {@link #weakValues()} or
071     * {@link #softValues()} was specified, the map uses identity comparisons
072     * for values.
073     *
074     * <p>The returned map has <i>weakly consistent iteration</i>: an iterator
075     * over one of the map's view collections may reflect some, all or none of
076     * the changes made to the map after the iterator was created.
077     *
078     * <p>An entry whose key or value is reclaimed by the garbage collector
079     * immediately disappears from the map. (If the default settings of strong
080     * keys and strong values are used, this will never happen.) The client can
081     * never observe a partially-reclaimed entry. Any {@link java.util.Map.Entry}
082     * instance retrieved from the map's {@linkplain Map#entrySet() entry set}
083     * is a snapshot of that entry's state at the time of retrieval; such entries
084     * do, however, support {@link java.util.Map.Entry#setValue}.
085     *
086     * <p>The maps produced by {@code MapMaker} are serializable, and the
087     * deserialized maps retain all the configuration properties of the original
088     * map. If the map uses soft or weak references, the entries will be
089     * reconstructed as they were, but there is no guarantee that the entries won't
090     * be immediately reclaimed.
091     *
092     * <p>{@code new MapMaker().weakKeys().makeMap()} can almost always be
093     * used as a drop-in replacement for {@link java.util.WeakHashMap}, adding
094     * concurrency, asynchronous cleanup, identity-based equality for keys, and
095     * great flexibility.
096     *
097     * @author Bob Lee
098     * @author Kevin Bourrillion
099     * @since 2 (imported from Google Collections Library)
100     */

185       * Specifies the maximum number of entries the map may contain. While the
186       * number of entries in the map is not guaranteed to grow to the maximum,
187       * the map will attempt to make the best use of memory without exceeding the
188       * maximum number of entries. As the map size grows close to the maximum,
189       * the map will evict entries that are less likely to be used again. For
190       * example, the map may evict an entry because it hasn't been used recently
191       * or very often.
192       *
193       * @param size the maximum size of the map
194       *
195       * @throws IllegalArgumentException if {@code size} is not greater than zero
196       * @throws IllegalStateException if a maximum size was already set
197       */

201        checkState(this.maximumSize == UNSET_MAXIMUM_SIZE,
202            "maximum size was already set to " + this.maximumSize);
203        checkArgument(size > 0, "maximum size must be positive");
204        this.maximumSize = size;
205        this.useCustomMap = true;
206        return this;
207      }

209      /**
210       * Guides the allowed concurrency among update operations. Used as a
211       * hint for internal sizing. The table is internally partitioned to try
212       * to permit the indicated number of concurrent updates without
213       * contention.  Because placement in hash tables is essentially random,
214       * the actual concurrency will vary. Ideally, you should choose a value
215       * to accommodate as many threads as will ever concurrently modify the
216       * table. Using a significantly higher value than you need can waste
217       * space and time, and a significantly lower value can lead to thread
218       * contention. But overestimates and underestimates within an order of
219       * magnitude do not usually have much noticeable impact. A value of one
220       * is appropriate when it is known that only one thread will modify and
221       * all others will only read. Defaults to 16.
222       *
223       * @throws IllegalArgumentException if {@code concurrencyLevel} is
224       *     nonpositive
225       * @throws IllegalStateException if a concurrency level was already set
226       */
227      @GwtIncompatible("java.util.concurrent.ConcurrentHashMap concurrencyLevel")
228      @Override
229      public MapMaker concurrencyLevel(int concurrencyLevel) {
230        checkState(this.concurrencyLevel == UNSET_CONCURRENCY_LEVEL,
231            "concurrency level was already set to " + this.concurrencyLevel);
232        checkArgument(concurrencyLevel > 0);
233        this.concurrencyLevel = concurrencyLevel;
234        return this;
235      }
236    
237      int getConcurrencyLevel() {
238        return (concurrencyLevel == UNSET_CONCURRENCY_LEVEL)
239            ? DEFAULT_CONCURRENCY_LEVEL : concurrencyLevel;
240      }

282      MapMaker setKeyStrength(Strength strength) {
283        checkState(keyStrength == null,
284            "Key strength was already set to " + keyStrength + ".");
285        keyStrength = checkNotNull(strength);
286        if (strength != Strength.STRONG) {
287          // STRONG could be used during deserialization.
288          useCustomMap = true;

290        return this;
291      }

298       * Specifies that each value (not key) stored in the map should be
299       * wrapped in a {@link WeakReference} (by default, strong references

360      /**
361       * Specifies that each entry should be automatically removed from the
362       * map once a fixed duration has passed since the entry's creation.
363       * Note that changing the value of an entry will reset its expiration
364       * time.
365       *
366       * @param duration the length of time after an entry is created that it
367       *     should be automatically removed
368       * @param unit the unit that {@code duration} is expressed in
369       * @throws IllegalArgumentException if {@code duration} is not positive
370       * @throws IllegalStateException if the expiration time was already set
371       */
372      @Override
373      public MapMaker expiration(long duration, TimeUnit unit) {
374        checkState(expirationNanos == UNSET_EXPIRATION_NANOS,
375            "expiration time of " + expirationNanos + " ns was already set");
376        checkArgument(duration > 0,
377            "invalid duration: " + duration);
378        this.expirationNanos = unit.toNanos(duration);
379        useCustomMap = true;
380        return this;
381      }
382    
383      long getExpirationNanos() {
384        return (expirationNanos == UNSET_EXPIRATION_NANOS)
385            ? DEFAULT_EXPIRATION_NANOS : expirationNanos;
386      }

388      /**
389       * Specifies a listener instance, which all maps built using this {@code
390       * MapMaker} will notify each time an entry is evicted.
391       *
392       * <p>A map built by this map maker will invoke the supplied listener after it
393       * evicts an entry, whether it does so due to timed expiration, exceeding the
394       * maximum size, or discovering that the key or value has been reclaimed by
395       * the garbage collector. It will invoke the listener synchronously, during
396       * invocations of any of that map's public methods (even read-only methods).
397       * The listener will <i>not</i> be invoked on manual removal.
398       *
399       * <p><b>Important note:</b> Instead of returning <em>this</em> as a {@code
400       * MapMaker} instance, this method returns {@code GenericMapMaker<K, V>}.
401       * From this point on, either the original reference or the returned
402       * reference may be used to complete configuration and build the map, but only
403       * the "generic" one is type-safe. That is, it will properly prevent you from
404       * building maps whose key or value types are incompatible with the types
405       * accepted by the listener already provided; the {@code MapMaker} type cannot
406       * do this. For best results, simply use the standard method-chaining idiom,
407       * as illustrated in the documentation at top, configuring a {@code MapMaker}
408       * and building your {@link Map} all in a single statement.
409       *
410       * <p><b>Warning:</b> if you ignore the above advice, and use this {@code
411       * MapMaker} to build maps whose key or value types are incompatible with the
412       * listener, you will likely experience a {@link ClassCastException} at an
413       * undefined point in the future.
414       *
415       * @since 7
416       */
417      @Beta
418      @GwtIncompatible("To be supported")
419      public <K, V> GenericMapMaker<K, V> evictionListener(
420          MapEvictionListener<K, V> listener) {
421        checkState(this.evictionListener == null);
422    
423        // safely limiting the kinds of maps this can produce
424        @SuppressWarnings("unchecked")
425        GenericMapMaker<K, V> me = (GenericMapMaker<K, V>) this;
426        me.evictionListener = checkNotNull(listener);
427        useCustomMap = true;
428        return me;
429      }

431      /**
432       * Builds a map, without on-demand computation of values. This method
433       * does not alter the state of this {@code MapMaker} instance, so it can be
434       * invoked again to create multiple independent maps.
435       *
436       * @return a serializable concurrent map having the requested features
437       */
438      @Override
439      public <K, V> ConcurrentMap<K, V> makeMap() {
440        return useCustomMap
441            ? new CustomConcurrentHashMap<K, V>(this)
442            : new ConcurrentHashMap<K, V>(getInitialCapacity(),
443                0.75f, getConcurrencyLevel());
444      }

446      /**
447       * Builds a caching function, which either returns an already-computed value
448       * for a given key or atomically computes it using the supplied function.
449       * If another thread is currently computing the value for this key, simply
450       * waits for that thread to finish and returns its computed value. Note that
451       * the function may be executed concurrently by multiple threads, but only for
452       * distinct keys.
453       *
454       * <p>The {@code Map} view of the {@code Cache}'s cache is only
455       * updated when function computation completes. In other words, an entry isn't
456       * visible until the value's computation completes. No methods on the {@code
457       * Map} will ever trigger computation.
458       *
459       * <p>{@link Cache#apply} in the returned function implementation may
460       * throw:
461       *
462       * <ul>
463       * <li>{@link NullPointerException} if the key is null or the
464       *     computing function returns null
465       * <li>{@link ComputationException} if an exception was thrown by the
466       *     computing function. If that exception is already of type {@link
467       *     ComputationException} it is propagated directly; otherwise it is
468       *     wrapped.
469       * </ul>
470       *
471       * <p>If {@link Map#put} is called on the underlying map before a computation
472       * completes, other threads waiting on the computation will wake up and return
473       * the stored value. When the computation completes, its new result will
474       * overwrite the value that was put in the map manually.
475       *
476       * <p>This method does not alter the state of this {@code MapMaker} instance,
477       * so it can be invoked again to create multiple independent maps.
478       *
479       * @param computingFunction the function used to compute new values
480       * @return a serializable cache having the requested features
481       */
482      // TODO(kevinb): figure out the Cache interface before making this public
483      <K, V> Cache<K, V> makeCache(
484          Function<? super K, ? extends V> computingFunction) {
485        return new ComputingConcurrentHashMap<K, V>(this, computingFunction);
486      }

488      /**
489       * Builds a map that supports atomic, on-demand computation of values. {@link
490       * Map#get} either returns an already-computed value for the given key,
491       * atomically computes it using the supplied function, or, if another thread
492       * is currently computing the value for this key, simply waits for that thread
493       * to finish and returns its computed value. Note that the function may be
494       * executed concurrently by multiple threads, but only for distinct keys.
495       *
496       * <p>If an entry's value has not finished computing yet, query methods
497       * besides {@code get} return immediately as if an entry doesn't exist. In
498       * other words, an entry isn't externally visible until the value's
499       * computation completes.
500       *
501       * <p>{@link Map#get} on the returned map will never return {@code null}. It
502       * may throw:
503       *
504       * <ul>
505       * <li>{@link NullPointerException} if the key is null or the computing
506       *     function returns null
507       * <li>{@link ComputationException} if an exception was thrown by the
508       *     computing function. If that exception is already of type {@link
509       *     ComputationException} it is propagated directly; otherwise it is
510       *     wrapped.
511       * </ul>
512       *
513       * <p><b>Note:</b> Callers of {@code get} <i>must</i> ensure that the key
514       * argument is of type {@code K}. The {@code get} method accepts {@code
515       * Object}, so the key type is not checked at compile time. Passing an object
516       * of a type other than {@code K} can result in that object being unsafely
517       * passed to the computing function as type {@code K}, and unsafely stored in
518       * the map.
519       *
520       * <p>If {@link Map#put} is called before a computation completes, other
521       * threads waiting on the computation will wake up and return the stored
522       * value. When the computation completes, its new result will overwrite the
523       * value that was put in the map manually.
524       *
525       * <p>This method does not alter the state of this {@code MapMaker} instance,
526       * so it can be invoked again to create multiple independent maps.
527       *
528       * @param computingFunction the function used to compute new values
529       * @return a serializable concurrent map having the requested features
530       */
531      @Override
532      public <K, V> ConcurrentMap<K, V> makeComputingMap(
533          Function<? super K, ? extends V> computingFunction) {
534        Cache<K, V> cache = makeCache(computingFunction);
535        return new ComputingMapAdapter<K, V>(cache);
536      }

538      /**
539       * A function which caches the result of each application (computation). This
540       * interface does not specify the caching semantics, but does expose a {@code
541       * ConcurrentMap} view of cached entries.
542       */
543      interface Cache<K, V> extends Function<K, V> {
544    
545        /**
546         * Returns a map view of the cached entries.
547         */
548        ConcurrentMap<K, V> asMap();
549      }

551      /**
552       * Overrides get() to compute on demand.
553       */
554      static class ComputingMapAdapter<K, V> extends ForwardingConcurrentMap<K, V>
555          implements Serializable {
556        private static final long serialVersionUID = 0;
557    
558        final Cache<K, V> cache;
559    
560        ComputingMapAdapter(Cache<K, V> cache) {
561          this.cache = cache;
562        }
563    
564        @Override protected ConcurrentMap<K, V> delegate() {
565          return cache.asMap();
566        }
567    
568        @SuppressWarnings("unchecked") // unsafe, which is why this is deprecated
569        @Override public V get(Object key) {
570          return cache.apply((K) key);
571        }
572      }
