nullAwareEquals

public static Equivalence<Object> nullAwareEquals()
Returns an equivalence that delegates to Object.equals(java.lang.Object) and Object.hashCode(). Equivalence.equivalent(T, T) returns true if both values are null, or if neither value is null and Object.equals(java.lang.Object) returns true. Equivalence.hash(T) returns 0 if passed a null value.

identity

public static Equivalence<Object> identity()
Returns an equivalence that uses == to compare values and System.identityHashCode(Object) to compute the hash code. Equivalence.equivalent(T, T) returns true if both values are null, or if neither value is null and == returns true. Equivalence.hash(T) throws a NullPointerException if passed a null value.
