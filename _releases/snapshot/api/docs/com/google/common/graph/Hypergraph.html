Interface Hypergraph<N,E>

    @Beta
public interface Hypergraph<N,E>
extends Graph<N,E>
    A subinterface of Graph which specifies that all edges are hyperedges, that is, they connect arbitrary sets of nodes rather than pairs of nodes.

    A few notes about how hyperedges and connectivity:

    • Hyperedges, like undirected edges, are both incoming and outgoing edges.
    • Hyperedges incident to a single node node connect node to itself; such edges are analogous to self-loops in graphs. Hyperedges incident to > 1 nodes do not connect any of their incident nodes to themselves.
