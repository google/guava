com.google.common.eventbus

Annotation Type Subscribe


  • @Retention(value=RUNTIME)
@Target(value=METHOD)
@Beta
public @interface Subscribe
    Marks a method as an event subscriber.

    The type of event will be indicated by the method's first (and only) parameter. If this annotation is applied to methods with zero parameters, or more than one parameter, the object containing the method will not be able to register for event delivery from the EventBus.

    Unless also annotated with @AllowConcurrentEvents, event subscriber methods will be invoked serially by each event bus that they are registered with.
