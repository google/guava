Interface ByteArrayDataOutput

  • All Superinterfaces:
    DataOutput

    public interface ByteArrayDataOutput
extends DataOutput
    An extension of DataOutput for writing to in-memory byte arrays; its methods offer identical functionality but do not throw IOException.
    Since:
    1.0
    Author:
