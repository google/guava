Class ForwardingMultiset.StandardElementSet

  • java.lang.Object
    • java.util.AbstractCollection<E>

    Enclosing class:
    ForwardingMultiset<E>

    @Beta
protected class ForwardingMultiset.StandardElementSet
extends AbstractSet<E>
    A sensible implementation of Multiset.elementSet() in terms of the following methods: ForwardingCollection.clear(), ForwardingCollection.contains(java.lang.Object), ForwardingCollection.containsAll(java.util.Collection<?>), ForwardingMultiset.count(java.lang.Object), ForwardingCollection.isEmpty(), the Set.size() and Set.iterator() methods of ForwardingMultiset.entrySet(), and ForwardingMultiset.remove(Object, int). In many situations, you may wish to override ForwardingMultiset.elementSet() to forward to this implementation or a subclass thereof.

    10.0
